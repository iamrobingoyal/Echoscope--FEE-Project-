# EchoScope 🌸

EchoScope is a cute, aesthetically pleasing productivity web application built for students. It helps you understand your digital habits and boost your focus with light gamification and a playful "doodle" UI.

## Features ✨

*   **📋 To-Do List:** Track your daily tasks with a satisfying checklist.
*   **📊 Analysis:** Log your screen time and reflect on your digital habits.
*   **⏳ Focus Timer:** Boost your productivity with Pomodoro sessions.
*   **⏱️ Stopwatch:** Track your active time accurately.

## Technologies Used 💻

*   HTML5
*   CSS3 (Custom doodle-style CSS, floating animations, and cute pastel gradients)
*   Vanilla JavaScript (Local storage for simulated login, interactive timers and tasks)

## Design Aesthetics 🎨

EchoScope features a unique, hand-drawn "kid doodle" aesthetic with:
*   Floating background elements and dynamic animations.
*   A vibrant pastel color palette (soft pinks, lavenders, and sky blues).
*   Playful typography using Google Fonts (Caveat and Nunito).

## Setup Instructions 🚀

1.  Clone this repository to your local machine.
2.  Open the project directory.
3.  Launch `index.html` in your favorite web browser, or use an extension like **Live Server** in VS Code for the best experience.
4.  No build steps or dependencies are required!

## Project Structure 📁

*   `index.html`: The main dashboard/home page.
*   `tasks.html` & `tasks.js`: The To-Do list functionality.
*   `analysis.html`: The screen time analysis page.
*   `timer.html` & `timer.js`: The Pomodoro focus timer.
*   `stopwatch.html` & `stopwatch.js`: The active time stopwatch.
*   `login.html`: A simulated login system saving progress locally.
*   `about.html`: Information about the project.
*   `styles.css`: All the styling and doodle animations.
*   `random-bg.js`: Scripts for dynamic background effects.
